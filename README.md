## 🚀 Quickstart

1. Clone the repo

```bash
git clone https://github.com/sarcastic910/junaid-s-cyber-codex.git
cd junaid-s-cyber-codex
```

2. Install dependencies

```bash
npm install
# or
# pnpm install
# or
# bun install
```

3. Start dev server

```bash
npm run dev
```

Open `http://localhost:5173` (or the URL shown in the terminal).
